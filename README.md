# USAGE

## AS A MODULE USING `BALE`.
```cpp
import Buffer "buffer"
Buffer buf(10);
```

## AS A REGLUAR C++ LIBRARY.
Build with `buffer.cc` as a dependency and include the header file.

```cpp
#include "buffer.h"
Buffer buf(10);
```
